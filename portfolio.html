About Me

Hi, I'm Pankaj Kumar Singh, a passionate Full Stack Developer with experience in building dynamic and responsive web applications using HTML, CSS, JavaScript, Python, and MySQL. I love creating clean, user-friendly interfaces and solving real-world problems through code.
